<h1>프로그래머스 자바스크립트 알고리즘</h1><br>
<h2>1. 문자열 내 마음대로 정렬하기</h2><br>


~~~
function solution(strings, n) {
    strings.sort((a,b)=>{
        if(a[n] == b[n]) solution(strings, ++n);
        return (!a[n] || !b[n])? 0 : +(a[n]>b[n]);
    }
    );
    
    return strings;
}
~~~
